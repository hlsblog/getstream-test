# Stream Feed 应用

基于 GetStream Activity Feeds API 构建的社交动态流应用。

## 功能特性

### ✅ 已实现功能

1. **用户连接** - 输入用户名连接到 Stream 服务
2. **动态列表** - 显示包含图片、描述、点赞数、评论数的活动列表
3. **排序功能** - 支持按时间和点赞数排序
4. **关注功能** - 关注/取消关注其他用户
5. **点赞功能** - 点赞/取消点赞动态
6. **评论功能** - 对动态进行评论
7. **响应式设计** - 支持移动端和桌面端

### 🎨 界面特色

- 现代化的渐变设计
- 流畅的动画效果
- 直观的用户交互
- 移动端友好的响应式布局

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **API**: GetStream Activity Feeds API
- **样式**: 原生CSS，渐变和动画效果
- **服务器**: Python HTTP Server (开发环境)

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动本地服务器

```bash
# 使用 Python
python3 -m http.server 8002

# 或使用 Node.js
npx http-server -p 8002
```

### 3. 访问应用

打开浏览器访问: `http://localhost:8002/index.html`

## 使用说明

### 连接用户
1. 在顶部输入框中输入用户名
2. 点击"连接"按钮或按回车键
3. 连接成功后状态指示器变为绿色

### 查看动态
- 动态列表会自动加载
- 每个动态包含用户头像、用户名、发布时间、图片、描述文字
- 显示点赞数和评论数

### 排序动态
- 使用排序下拉菜单选择排序方式
- 支持"按时间排序"和"按点赞数排序"

### 关注功能
- 连接成功后会显示"关注"按钮
- 点击可关注/取消关注演示用户
- 按钮状态会相应更新

### 互动功能
- **点赞**: 点击心形图标进行点赞/取消点赞
- **评论**: 点击评论图标展开评论区，输入评论后点击发送

## API 集成

应用集成了 GetStream Activity Feeds API，支持：

- 用户认证和连接
- 活动流获取
- 关注关系管理
- 活动互动（点赞、评论）

### 演示模式

当前应用运行在演示模式下，使用模拟数据展示功能。在生产环境中需要：

1. 配置真实的 Stream API 密钥
2. 实现服务器端用户令牌生成
3. 连接真实的用户数据库

## 开发说明

- 图片上传

- 用户搜索

## 注意事项

2. 生产环境需要配置真实的 Stream 账户和 API 密钥
